# WDCC React Workshop Examples Repository
This repository contains example code demonstrating various React features.

## Index

- *[Vite+React](./example-01-vite-default)*: Shows the default output of running `npm create vite@latest`, selecting `React` and `JavaScript + SWC` as options, as of February 2023.

- *[Components, Logic & Loops](./example-02-components-logic-loops)*: Demonstrates simple React components with props, use of the ternary operator (`?`) and `&&` for conditional rendering, and iteration using the `map()` function.

- *[CSS Imports](./example-03-css-imports)*: Demonstrates how to import and use raw CSS files in react webapps.

- *[CSS Modules](./example-04-css-modules)*: Demonstrates how to import and use CSS modules.

- *[useState() Hook](./example-05-usestate)*: Demonstrates how to give components local state using the `useState()` hook.

- *[useEffect() Hook](./example-06-useeffect)*: Demonstrates how to have components give side-effects using the `useEffect()` hook.

- *[Routing Examples](./example-07-routing-01)*: Demonstrates basic use of React Router, **version 6** *(NOT backwards compatible with older versions)*.

- *[More Routing Examples](./example-08-routing-02)*: Demonstrates mode advanced use of React Router.

- *[Consuming APIs](./example-09-api-calls)*: Demonstrates how we can use the axios library to make HTTP requests to our backend APIs.
